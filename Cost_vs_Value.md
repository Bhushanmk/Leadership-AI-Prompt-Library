### TASK
Create a **leadership-ready cost vs value justification** for an initiative or investment.

### CONTEXT
- Audience: Executives, Finance, Clients
- Goal: Support funding or approval decisions
- Style: Business-first, risk-aware

### RULES
1. Frame value in outcomes, not features
2. Include cost of inaction
3. Avoid over-promising ROI
4. Keep it concise and balanced

### CLARIFICATION RULE
Ask before proceeding if unclear:
- Approval context (budget / headcount / vendor)
- Time horizon (quarter / annual)
- Financial sensitivity
